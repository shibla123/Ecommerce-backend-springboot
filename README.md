# Ecommerce Backend (Spring Boot)

A RESTful backend for an e-commerce application built using Spring Boot and MySQL.

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Postman

## Features
- User Management APIs
- Product Management APIs
- Create Product
- Get All Products
- Get Product by ID
- Update Product
- Delete Product
- MySQL Database Integration

## API Endpoints

### Products
POST /api/products
GET /api/products
GET /api/products/{id}
PUT /api/products/{id}
DELETE /api/products/{id}

## Project Status
Backend development is in progress with additional features like authentication, cart, orders, and JWT security planned.
